## Installing

First, you'll need [node.js](https://nodejs.org/), which includes `npm`. Then, open the project folder on the terminal and run:

`$ npm i`

## Building and running

### Building

`$ gulp build`

### Watching

`$ gulp watch`

### Running

`$ gulp server`

Then open `http://localhost:8080/`.
